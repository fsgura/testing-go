# TESTING-GO

<b>Testing OS</b>: <i>linux</i>
<b>Distribution</b>: <i>Fedora 25</i>
<b>Workspace</b>: Go 1.7 installed using
</br>
<code>dnf -y install go-golang</code>
</br>
and dependencies from the standard fedora repo
